1：alt + 上下键—— 移动一行代码

2：ctrl + ./ctrl + 1—— 帮助

3：ctrl + shift + L—— 查看组合键

4：ctrl + 鼠标左键—— 查看变量、方法、类的定义

5：ctrl + T —— 查看接口的实现类

6：ctrl + shift + F—— 格式化选中的代码

7：ctrl + shift + o—— 缺少的import语句被导入，多余的import语句被删除

8：ctrl + shift + m—— 导入import语句

9：ctrl + shift + s—— 保存所有未保存的文件
