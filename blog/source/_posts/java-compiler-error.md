错误原因 eclipse 报错：

java complier level does not match the version of the installed java project

解决办法：

1.右键项目“Properties”，在弹出的“Properties”窗口左侧，单击“Project Facets”，打开“Project Facets”页面

2.在页面中的“Java”下拉列表中，选择相应版本
